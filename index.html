Game Synopsis

Honkai Impact 3rd is a F2P (Free to Play) action role-playing game made by miHoYo (Hoyoverse)

Honkai Impact 3rd is an action game that is set in a post-apocalyptic world where you play as various characters called Valkyries and each of them have various weapons and skills to fight enemies
